# Daily Task Tracker

A Production-ready Kanban-style task management application built with Vite and React.

## Features

- **Kanban Board**: Visualize tasks in 'To Do', 'In Progress', and 'Done' columns.
- **Task Management**: Create tasks with priority, category, and estimated time.
- **Interactive**: Move tasks between states and delete them.
- **Local Storage**: Data persists in your browser.
- **Responsive Design**: Works on desktop and mobile.
- **Forest Theme**: specific aesthetic design.

## Tech Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS (via CDN for speed/simplicity in this demo, but structured for scalablity)
- **Icons**: Lucide React

## Development

```bash
npm install
npm run dev
```

Server runs on `0.0.0.0:8080` by default.
